## 🚀 Getting Started

### Prerequisites

No build tools required — just a modern web browser!

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open the HTML file
   open index.html
   # Or use a local server (recommended)
   npx serve .
   ```

3. **Start customizing!**
   - Edit `index.html` for content changes
   - Modify `styles.css` to tweak the design
   - Update `script.js` for interaction changes

### Deployment

This is a static site — deploy anywhere:

| Platform | Command / Steps |
|----------|----------------|
| **GitHub Pages** | Push to `main` → Settings → Pages → Deploy from branch |
| **Netlify** | Drag & drop the folder, or connect your repo |
| **Vercel** | Import the repo → auto-deploys on push |

## 🎨 Customization

### Colors
All design tokens are defined as CSS custom properties in `styles.css`:

```css
:root {
    --accent-primary: #6c63ff;     /* Main purple accent */
    --accent-secondary: #00d4ff;   /* Cyan accent */
    --accent-pink: #ff6b9d;        /* Pink accent */
    --bg-primary: #0a0a0f;         /* Background */
}
```

### Content
Update these sections in `index.html`:
- **Hero** — Name, roles (typewriter array in `script.js`), description
- **About** — Bio, education, code window
- **Skills** — Languages, tools, interests
- **Projects** — Project cards with descriptions
- **Certifications** — Certificates & achievements
- **Contact** — Email, social links
